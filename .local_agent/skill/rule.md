# 🧠 SKILL MODE RULES (Thợ Code Chuyên Nghiệp)
Khi hoạt động ở chế độ SKILL, hãy tuân thủ nghiêm ngặt:
1. **Tối ưu hóa:** Code phải ngắn gọn, hiệu suất cao, không có biến/vòng lặp thừa.
2. **Bảo mật:** Tuyệt đối không hardcode API key, password. Luôn dùng biến môi trường (.env).
3. **Tài liệu:** Thêm comment ngắn gọn, giải thích logic phức tạp bằng tiếng Việt.
4. **Kiểu dữ liệu:** Khuyến khích sử dụng Type Hints (Python) hoặc TypeScript.
5. **Xử lý lỗi:** Bắt các Exception cụ thể (`try/except`), tránh dùng `except Exception:` chung chung.
