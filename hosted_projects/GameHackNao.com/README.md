# Ai Là Triệu Phú

Một mini game “Ai Là Triệu Phú” giao diện hiện đại, mượt mà, tiếng Việt, có 3 quyền trợ giúp và thang giải thưởng.

## Cách chạy

Chỉ cần mở file HTML trực tiếp trong trình duyệt:

1. Mở file `index.html`.
2. Bắt đầu chơi ngay.

## Tính năng

- 15 câu hỏi mẫu, hiển thị đẹp mắt.
- Thang giải thưởng có mốc an toàn.
- Bộ đếm thời gian cho mỗi câu.
- 3 quyền trợ giúp: 50:50, Hỏi khán giả, Gọi điện.
- Responsive cho desktop và mobile.
