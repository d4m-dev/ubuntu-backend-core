Chào bạn, để cài đặt Ollama (hệ thống nền tảng giúp chạy các mô hình AI trực tiếp trên máy của bạn mà không cần Internet), bạn hãy làm theo các bước chi tiết dưới đây:

Bước 1: Cài đặt Ollama
Nếu bạn đang dùng Linux hoặc môi trường máy chủ (như Ubuntu trên Termux/Android): Mở Terminal lên và chạy lệnh sau để tải và cài đặt tự động:

bash
curl -fsSL https://ollama.com/install.sh | sh
Nếu bạn đang dùng máy tính Windows hoặc macOS: Hãy truy cập trang web chính thức của họ tại https://ollama.com/download. Tải file cài đặt về và cài (Next -> Next) giống như các phần mềm bình thường khác.

Bước 2: Khởi động máy chủ Ollama
Sau khi cài đặt xong, bạn cần bật server của Ollama lên để nó lắng nghe các yêu cầu từ file code Python của bạn (ở cổng 11434). Hãy mở một tab Terminal (cửa sổ dòng lệnh) mới và gõ:

bash
ollama serve
(Lưu ý: Máy chủ này cần được giữ chạy liên tục. Bạn hãy thu nhỏ hoặc để nguyên tab Terminal này và mở một tab khác để làm các bước tiếp theo).

Bước 3: Tải "Bộ não" (Model AI) cho Ollama
Ollama giống như một cái máy phát nhạc, bạn cần phải có băng đĩa (Models) thì nó mới hát được. Theo như file code local_agent.py của bạn, hệ thống đang được cài đặt mặc định gọi tới model deepseek-coder:1.3b.

Tại tab Terminal mới, bạn hãy chạy lệnh sau để tải model này về máy:

bash
ollama pull deepseek-coder:1.3b
(Chỉ bước tải model này là cần kết nối Internet. Model này nặng khoảng chưa tới 1GB. Tải xong một lần là sau này bạn có thể rút mạng ra xài thoải mái).

Bước 4: Chạy Local Agent của bạn
Khi model đã báo tải thành công (success), bây giờ bạn đã có một AI Lập trình hoàn toàn offline. Bạn trỏ tới thư mục chứa file AI của bạn và chạy script:

bash
cd /sdcard/coder/media/AI-Local
python3 local_agent.py
